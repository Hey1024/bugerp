### 项目优势  
 1， 部署简单  
 2， 因为后端是go语言写的，跨平台，速度快（虽然这个没什么卵用的样子）  
 3， vue平台用的vue-element-admin框架上写的，支持手机端  
 4， 有需求就有更新  
 5， 永久开源  
 6， 后期后端代码会开源  
 7， 会有专门的接口文档，方便适应各种后端语言  
 8， 问题反馈迅速，作者不是程序员，不需要加班加点敲公司的代码
